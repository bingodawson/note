skip-grant-tables

mysql忘记密码：

use mysql;
update user set authentication_string = password("bluezhai") where user = 'root';

去掉skip-grant-tables，重启mysql服务。



用户管理，为团队成员分配合适的权限
create user one@localhost identified by '123456';

git bash 里连接mysql，命令卡主不动的原因：
winpty   mysql -uroot -p
这样就直接能连上了,同样的使用其他的windows命令也是一样的

grant all privileges on *.* to one@localhost
grant all privileges on *.* to 'one'@localhost
用户名带不带单引号，无影响。
